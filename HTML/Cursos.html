Nossos Cursos

HTML e CSS

HTML é uma ("Linguagem de Marcação de Hipertexto"), utilizada para construir páginas WEB, Documentos de HTML podem ser interpretados por navergadores.


Css é chamado de linguagem Cascading Style Shee ("Planilha em Estilo Cascata"), e é utilizado para estilizar uma liguagem de marcação, tal como o HTML, a dirença entre o HTML e o CSS é que o, HTML é a parte estrutural do site, nele vai caixas de conteúdo, textos..., Já o CSS é a parte de estilização, para deixar todos os componentes do HTML, mas visíveis aos olhos, geralmente no CSS, adicionamos cores, imagens, e alteramos os comprimentos dos componentes.

Sistemas Operacionais

Sistemas Operacionais (Operating System) é um programa ou conjunto de programas cuja a função é gerenciar os recursos do sistema, basicamente os "SO", tem como objetivo deixar mas fácil a interação entre o computador e os usuário


Os Sistemas Operacionais mas usados são (Android, Windows, IOS, Linux...).
